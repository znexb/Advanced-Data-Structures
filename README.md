# How to run each file in assignments?

This how-to is for Windows.

! Requires git (installed and added to PATH)
! Powershell recommended


## Steps

- Open up Powershell (optionally as Admin) and cd to the desired extraction point for this project.
- Run the following command to _clone_ the repository: "git clone "https://github.com/znexb/Advanced-Data-Structures.git"".
- Run the following command to _run_ the desired program: "./{PROGRAM_NAME}.exe", 
  - for example for Binary Search Trees, one would replace {PROGRAM_NAME} with BST -> "./BST.exe".

## Note

- There is currently no GUI, thus one cannot directly select which values to insert. 
- To modify values and experiment, you have to modify the corresponding .c file.
